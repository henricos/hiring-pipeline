# Hiring Pipeline

Ferramenta interna de contratação para apoiar a abertura de vagas, gestão de candidatos e entrevistas, e acelerar avaliações com apoio de IA.

O foco inicial do projeto é organizar e simplificar o processo de recrutamento da área, desde a definição do perfil da vaga até a decisão final sobre os candidatos.
