Criar arquivo src/lib/repositories/research-repository.ts com implementação exata do padrão ProfileRepository, adaptado para pesquisas.

Estrutura da implementação:

1. **Interface ResearchRepository**:
```typescript
export interface Research {
  profileId: string;
  date: string; // YYYY-MM-DD
  baseName: string; // data ou data-2 (sufixo opcional)
  vagasFile: string; // {date}-vagas.json
  resumoFile: string; // {date}-resumo.json
}

export interface ResearchRepository {
  listByProfileId(profileId: string): Promise<Research[]>;
  getVagas(profileId: string, date: string): Promise<any | null>;
  getResumo(profileId: string, date: string): Promise<any | null>;
}
```

2. **Classe JsonResearchRepository implements ResearchRepository**:

**Método researchPath(profileId: string): string**
- Validar profileId: rejeitar strings vazias, "..", "/", "\\"
- Lançar Error com mensagem "ID de perfil inválido: "{profileId}"" se validação falhar
- Chamar ensureSubdir("research") para obter diretório base
- Retornar path.join(dir, profileId)

**Método listByProfileId(profileId: string): Promise<Research[]>**
- Chamar this.researchPath(profileId) para obter diretório
- Envolver em try/catch — retornar [] se directory não existe (ENOENT)
- Usar fs.readdirSync(dir) para obter arquivos
- Filtrar apenas arquivos .json
- Usar regex `^(\d{4}-\d{2}-\d{2})(?:-\d)?-(vagas|resumo)\.json$` para extrair data e tipo
  - Grupo 1: data (YYYY-MM-DD)
  - Grupo 2 (opcional): sufixo (-2, -3, etc.)
  - Grupo 3: tipo (vagas ou resumo)
- Agrupar por data usando Map<string, Research>:
  - Se data não existe no map, criar novo Research com profileId, date, baseName=""
  - Se vagasFile encontrado, atualizar research.vagasFile
  - Se resumoFile encontrado, atualizar research.resumoFile
- Converter Map em array e ordenar por date decrescente (mais recente primeiro)
- Retornar array ordenado

**Método getVagas(profileId: string, date: string): Promise<any | null>**
- Chamar this.researchPath(profileId)
- Construir nome do arquivo: `${date}-vagas.json`
- Envolver em try/catch
- Verificar fs.existsSync(filePath) — retornar null se não existe
- Ler fs.readFileSync(filePath, "utf-8") e fazer JSON.parse()
- Retornar objeto parseado
- Retornar null em caso de erro

**Método getResumo(profileId: string, date: string): Promise<any | null>**
- Idêntico a getVagas, mas com `${date}-resumo.json`

3. **Singleton export**:
```typescript
export const researchRepository: ResearchRepository = new JsonResearchRepository();
```

Padrão de error handling: sempre return [] ou null em catch, nunca lançar exceptions (silent fallback).

Todos os métodos devem ser async (mesmo que usem fs síncrono).

import { researchRepository } from "@/lib/repositories/research-repository";

}

  profileId: string,
  date: string
): Promise<any | null> {
  return researchRepository.getResumo(profileId, date);
}
