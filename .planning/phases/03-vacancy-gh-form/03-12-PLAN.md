<objective>
Mover campos fixos por área de Vacancy/JobProfile para AreaSettings — schema, formulários e gerador Excel.

Purpose: Vários campos têm valor fixo por área (a mesma equipe sempre usa o mesmo centro de custo, modalidade, horário, nível de inglês, etc.). Preenchê-los em cada perfil/vaga é retrabalho. Mover para Configurações da Área significa preencher uma vez e reutilizar automaticamente.

Campos movendo de JobProfile → AreaSettings:
- englishLevel, spanishLevel, otherLanguage, otherLanguageLevel
- additionalInfo (informações complementares)
- systemsRequired, networkFolders (infraestrutura)

Campos movendo de Vacancy → AreaSettings:
- costCenter, workMode, travelRequired
- workSchedule, workScheduleOther (horário de trabalho)

Estratégia de retrocompatibilidade: JobProfile e Vacancy mantêm os campos como opcionais (?) nos tipos TypeScript, para que JSONs existentes continuem sendo lidos sem erro. Os formulários simplesmente param de exibir e submeter esses campos — os values existentes no JSON são preservados mas não mais editáveis (ficam obsoletos).

Output: AreaSettings expandida + formulários de settings com novos campos + excel-generator lendo de settings.

export async function updateSettings(...): Promise<...> {

    const immediateReport = (formData.get("immediateReport") as string) || "";

    const spanishLevel = (formData.get("spanishLevel") as string) as AreaSettings["spanishLevel"] || "Não exigido";
    const otherLanguage = (formData.get("otherLanguage") as string) || undefined;
    const otherLanguageLevel = (formData.get("otherLanguageLevel") as string) as AreaSettings["otherLanguageLevel"] || undefined;
    // Conteúdo
    const additionalInfo = (formData.get("additionalInfo") as string) || "";
    // Infraestrutura
    const systemsRequired = (formData.get("systemsRequired") as string) || undefined;
    const networkFolders = (formData.get("networkFolders") as string) || undefined;
    // Dados da vaga
    const costCenter = (formData.get("costCenter") as string) || "";
    const workSchedule = ((formData.get("workSchedule") as string) || "Das 08h às 17h") as AreaSettings["workSchedule"];
    const workScheduleOther = workSchedule === "Outro"
      ? (formData.get("workScheduleOther") as string) || undefined
      : undefined;
    const travelRequired = formData.get("travelRequired") === "true";
    const workMode = ((formData.get("workMode") as string) || "Presencial") as AreaSettings["workMode"];

      englishLevel, spanishLevel, otherLanguage, otherLanguageLevel,

      costCenter, workSchedule, workScheduleOther, travelRequired, workMode,
    };

    revalidatePath("/settings");

}
